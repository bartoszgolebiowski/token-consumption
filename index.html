Extensibility

Adding New Slides Is Simple

Just add a new HTML file under the pages/ directory. The build compiler automatically wraps it and injects it into your deck.

pages/06-my-slide.html 
<!-- transition: zoom, layout: slide-layout-title -->
        <h2>My New Slide Title</h2>
        <p>Exciting slide body content goes here...</p>
        <div class="notes">
            Presenter notes visible in speaker mode.
        </div>
Show how clean the markup is now that the wrapping section has been removed. Explain that they can customize transitions and layouts using comments.

⌨️ Shortcuts

  • Space / Right Arrow: Next Slide
  • Left Arrow: Previous Slide
  • O Key: Toggle Overview Mode
  • T Key: Cycle Styling Themes
  • N Key: Toggle Speaker Notes Drawer
  • F Key: Toggle Fullscreen Mode

📁 Project Structure

  • pages/: Place your minimal HTML files here
  • theme.css: Change presentation coloring and themes
  • style.css: Core layouts and navigation drawer styles
  • template.html: The presentation shell template
